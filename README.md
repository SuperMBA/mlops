# MLOps HW1 — Iris classification

## Цель проекта

Построить минимальный, но воспроизводимый ML-пайплайн на примере датасета Iris с использованием:
- Git для контроля версий кода,
- DVC для версионирования данных и описания пайплайна,
- MLflow для логирования экспериментов (параметры, метрики, артефакты).

## Структура проекта

```bash
.
├── data/                # данные (сырые и обработанные) — под управлением DVC
│   ├── raw/             # сырые данные (iris.csv)
│   └── processed/       # train/test после prepare
├── src/
│   ├── prepare.py       # подготовка данных
│   └── train.py         # обучение модели и логирование в MLflow
├── dvc.yaml             # описание DVC-пайплайна (stages: prepare, train)
├── dvc.lock             # зафиксированные версии стадий
├── params.yaml          # гиперпараметры разбиения и модели
├── requirements.txt     # зависимости проекта
└── README.md
